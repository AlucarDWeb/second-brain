## How to use this vault
1. Set your domain in `CLAUDE.md` (one-time).
2. Drop sources into `Clippings/`.
3. Run `/compile` to build Sources + Concepts + Maps.
4. Run `/ask <question>` to query the vault (cited, vault-only).
5. Run `/lint` periodically for a health check.
